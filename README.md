# 📝 BLOG - Full-Stack Blogging Platform

**Theme**: Connection and Creativity  
**Motto**: Unleash voices, spark conversations.

---

## 🌐 Overview

**BLOGIFY** is a modern, full-stack blogging platform designed to foster connections between readers and writers. With built-in content discovery, real-time interactions, and a clean, intuitive interface, users can explore trending posts, follow favorite authors, and share their own stories effortlessly.

---

## 💡 Idea

BLOGIFY empowers both creators and readers through an interactive ecosystem:

- **Content Discovery**: Readers browse popular posts and top authors based on engagement, making it easy to find fresh perspectives.
- **Markdown-Powered Writing**: Authors craft posts using a rich-text, markdown-supported editor with seamless image uploads via Cloudinary.
- **Community Engagement**: A dynamic comment system enables posting, editing, and deleting comments in real time.
- **Personalized Feeds**: Follow or unfollow authors to curate your own reading experience.
- **Secure Authentication**: Google OAuth 2.0 integration ensures fast and secure logins.

## 🚀 Key Features

- 🔍 **Content Discovery**  
  Explore blogs and view top authors based on engagement metrics.

- ✍️ **Blog Creation**  
  Built-in markdown editor with image upload support via Cloudinary.

- 💬 **Comment System**  
  Post, edit, and delete comments with real-time synchronization via Socket.IO.

- 👥 **Follow System**  
  Follow/unfollow authors to personalize user feeds.

- 🔑 **Authentication**  
  Google OAuth 2.0 for quick and secure login.

---

## 🛠️ Tech Stack

- **Frontend**: React.js, Tailwind CSS, Material UI  
- **State Management**: Redux  
- **Real-Time**: Socket.IO  
- **Backend**: Node.js, Express.js  
- **Database**: MongoDB  
- **Image Storage**: Cloudinary  
- **Authentication**: Google OAuth 2.0
